# 🎮 지긁재긁 (Jigeul-Jaegeul)

> **긁히면 진다.**

10분 동안 몰려오는 **스트레스와 밈**을 뚫고 두 보스를 쓰러뜨리는 웹 로그라이크 생존 게임.

---

## 📖 소개

**지긁재긁**은 누구나 한 번쯤 겪어본 "긁히는 순간"들을 게임으로 만든 웹 게임입니다.

"ㅋㅋ"

"5분만."

"회의 하나만."

"배송 지연"

"리뷰 ★1"

"품절"

평범한 말들이 몬스터가 되어 몰려옵니다.

당신은 끝까지 버틸 수 있을까요?

---

## 🎯 게임 목표

* 5분 중간 보스와 10분 최종 보스를 쓰러뜨리세요.
* 레벨업으로 일반·희귀·에픽·전설 능력을 최대 3단계까지 성장시키세요.
* 보스가 떨어뜨린 보물상자에서 전설 능력을 획득하세요.
* 친구와 최고 기록을 경쟁하세요.

> **긁히면 지고, 버티면 승리합니다.**

---

## ⚔️ 게임 특징

### 🧠 자동 공격

공격은 자동으로 이루어집니다.

플레이어는 이동과 생존에만 집중하면 됩니다.

### 🎵 네온 신스 사운드

첫 입력부터 절차형 BGM이 재생되며 메뉴, 전투, 일시정지, 결과 화면에 따라 분위기가 달라집니다. 공격·타격·처치·피격·경험치·레벨업에도 전용 효과음이 적용됩니다.

---

### 📈 로그라이크 성장

레벨업마다 세 장의 능력 중 하나를 선택합니다. 능력은 일반·희귀·에픽·전설의 네 등급이며, 모든 능력은 3레벨이 최대입니다. 최대 레벨이 된 능력은 더 이상 후보에 나오지 않습니다.

등급은 생존 시간에 따라 일반(시작), 희귀(30초), 에픽(60초), 전설(90초) 순으로 해금됩니다.

예시

* ☕ 커피
* 🛡 강철 멘탈
* 🔥 팩트 폭격
* 📦 무료배송
* 😐 읽씹
* 💥 지긁재긁

매 판마다 다른 빌드를 만들 수 있습니다.

---

### 👾 개성 있는 적

평범한 몬스터가 아닙니다.

* "ㅋㅋ"
* "ㅇㅋ"
* "5분만"
* "회의 추가"
* "배송 지연"
* "쿠폰 종료"
* "품절"
* "재입고 미정"

5분에는 중간 보스, 10분에는 최종 보스가 등장합니다. 보스전 중에는 게임 시간이 멈추며 일반 적이 추가 생성되지 않습니다.

보스는 조준 사격과 회전 원형 탄막을 사용합니다. 공격 전 경고를 보고 피해야 하며, 체력이 절반 이하가 되면 격노해 패턴이 더 빠르고 촘촘해집니다.

## 🚀 플레이

[GitHub Pages에서 지긁재긁 플레이하기](https://juglee0527.github.io/jigeul-jaegeul/)

로컬에서는 다음 명령으로 실행할 수 있습니다.

```bash
npm install
npm run dev
```

프로덕션 빌드와 미리보기:

```bash
npm run build
npm run preview
```

Node.js 22 이상과 npm을 권장합니다.

### 조작 방법

- `WASD` 또는 방향키: 이동
- 공격: 가장 가까운 적을 향해 자동 발사
- 메인 메뉴: `↑`·`↓`로 항목 이동, `Enter`로 선택
- 일시정지: `ESC`, 메뉴는 `↑`·`↓`로 이동하고 `Enter`로 선택
- 능력 선택: `←`·`→`로 이동하고 `Enter`로 선택
- `Enter`: 결과 화면에서 다시 시작

현재 10분 게임 루프, 두 차례 보스전, 전설 보물상자, 4등급 성장, 점수와 브라우저 최고 기록 저장까지 구현되어 있습니다.

메뉴의 **오늘의 도전**에서는 로컬 날짜를 시드로 사용합니다. 같은 날짜에는 적 등장과 능력 후보 순서가 재현되며 날짜별 최고 점수가 저장됩니다.
