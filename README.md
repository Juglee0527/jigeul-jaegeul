# 🎮 지긁재긁 (Jigeul-Jaegeul)

> **긁히면 진다.**

끝없이 몰려오는 **스트레스와 밈** 속에서 살아남는 웹 로그라이크 생존 게임.

---

## 📖 소개

**지긁재긁**은 누구나 한 번쯤 겪어본 "긁히는 순간"들을 게임으로 만든 웹 게임입니다.

"ㅋㅋ"

"5분만."

"회의 하나만."

"배송 지연"

"리뷰 ★1"

"품절"

평범한 말들이 몬스터가 되어 몰려옵니다.

당신은 끝까지 버틸 수 있을까요?

---

## 🎯 게임 목표

* 끝없이 등장하는 몬스터를 처치하며 살아남으세요.
* 레벨업을 통해 다양한 능력을 획득하세요.
* 더 오래 생존하고 더 높은 점수를 기록하세요.
* 친구와 최고 기록을 경쟁하세요.

> **긁히면 지고, 버티면 승리합니다.**

---

## ⚔️ 게임 특징

### 🧠 자동 공격

공격은 자동으로 이루어집니다.

플레이어는 이동과 생존에만 집중하면 됩니다.

---

### 📈 로그라이크 성장

레벨업마다 새로운 능력을 선택합니다.

예시

* ☕ 커피
* 🛡 강철 멘탈
* 🔥 팩트 폭격
* 📦 무료배송
* 😐 읽씹
* 💥 지긁재긁

매 판마다 다른 빌드를 만들 수 있습니다.

---

### 👾 개성 있는 적

평범한 몬스터가 아닙니다.

* "ㅋㅋ"
* "ㅇㅋ"
* "5분만"
* "회의 추가"
* "배송 지연"
* "쿠폰 종료"
* "품절"
* "재입고 미정"

그리고...

최종 보스는 언제나 예상치 못한 곳에서 등장합니다.

---

### 🏆 점수 경쟁

* 생존 시간
* 처치 수
* 최고 콤보
* 최종 점수

친구들과 기록을 경쟁해 보세요.

## 🚀 플레이

현재 개발 중입니다. 로컬에서 다음 명령으로 실행할 수 있습니다.

```bash
npm install
npm run dev
```

프로덕션 빌드와 미리보기:

```bash
npm run build
npm run preview
```

Node.js 22 이상과 npm을 권장합니다.

### 조작 방법

- `WASD` 또는 방향키: 이동
- 공격: 가장 가까운 적을 향해 자동 발사
- `Enter`: 메뉴에서 게임 시작, 결과 화면에서 다시 시작

현재 메뉴, 기본 전투, 경험치 획득, 레벨업 능력 선택, 게임 오버와 재시작까지 구현되어 있습니다.
