# 제품 요구사항

## 1. 제품 정의

**지긁재긁(Jigeul-Jaegeul)**은 10분 동안 몰려오는 스트레스와 인터넷 밈을 피하고 자동으로 공격하며 두 보스를 쓰러뜨리는 웹 로그라이크 생존 게임이다.

- 대표 문구: `긁히면 진다.`
- 브라우저 제목: `지긁재긁 | 긁히면 진다`
- 플랫폼: 데스크톱 웹 브라우저 우선
- 배포: GitHub Pages
- 설치, 로그인, 서버 없이 플레이 가능

장르의 자동 공격·생존·경험치·레벨업 선택 구조만 활용하며 기존 게임의 그래픽, 캐릭터, 사운드, 명칭은 복제하지 않는다.

## 2. 제품 목표

- 링크 하나로 즉시 플레이할 수 있다.
- 짧게 접속해도 한 판의 성장과 결과를 경험할 수 있다.
- 최고 기록과 오늘의 도전 기록으로 반복 플레이를 유도한다.
- 적, 능력, 웨이브를 데이터 중심으로 쉽게 추가할 수 있다.
- 공개 저장소에서 단계별 개발 상태를 파악할 수 있다.

## 3. 핵심 사용자 경험

플레이어는 이동에 집중하고 공격은 자동으로 처리된다. 적을 처치해 성장하고, 레벨업 때마다 무작위 능력 세 개 중 하나를 선택한다. 5분 중간 보스와 10분 최종 보스를 처치하고 전설 보상을 선택하면 승리한다. 사망 또는 승리 후 결과와 최고 기록을 확인하고 즉시 다시 시작할 수 있어야 한다.

## 4. 1차 MVP 범위

### 포함

- 메뉴와 일반 플레이/오늘의 도전 선택
- WASD 및 방향키 이동
- 기본 적 생성과 추적
- 가장 가까운 적 대상 자동 공격
- 적 처치, 플레이어 피격, 게임 오버
- 경험치 아이템, 레벨, 능력 3개 선택
- 일반·희귀·에픽·전설 4등급, 시간별 해금, 능력별 최대 3레벨
- 시간 기반 난이도 증가
- 5분·10분 보스, 보스전 시간 정지, 전설 보물상자
- 생존 시간, 처치 수, 보스 처치 수, 레벨, 최종 점수
- LocalStorage 최고 기록과 일일 기록
- 시드 기반 난수
- 결과 화면과 재시작
- GitHub Pages 자동 배포

### MVP 이후

- 보스별 고유 공격 패턴
- 돌진형과 탱커형을 포함한 콘텐츠 확장
- 모바일 가상 조이스틱
- 사운드와 고급 화면 효과
- 온라인 랭킹, 로그인, 서버
- 상점, 결제, 실시간 멀티플레이

## 5. 성공 조건

- 처음 방문한 사용자가 별도 설명 없이 게임을 시작하고 이동할 수 있다.
- 시작부터 10분 최종 보스 승리 또는 사망, 결과 확인, 재시작까지 끊김 없이 동작한다.
- 같은 날짜의 오늘의 도전에서 동일한 랜덤 결과가 재현된다.
- 새로고침 후에도 최고 기록과 설정이 유지된다.
- 기준 해상도 1280×720에서 UI가 겹치지 않는다.
- 프로덕션 빌드가 오류 없이 완료되고 GitHub Pages에서 실행된다.

## 6. 비기능 요구사항

- TypeScript의 `any` 사용을 최소화한다.
- 적 수에 상한을 두고 화면 밖 오브젝트를 정리한다.
- 적, 능력, 웨이브와 점수 공식은 실행 코드에서 분리한다.
- 모든 변경 후 타입 검사와 프로덕션 빌드를 통과한다.
- 외부 이미지 없이도 Phaser Graphics와 텍스트만으로 플레이 가능해야 한다.
