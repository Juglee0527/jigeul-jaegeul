# 개발 로드맵

각 단계는 완료 후에도 실행 가능한 상태를 유지한다. 체크박스는 코드와 검증 결과가 모두 충족됐을 때만 완료 처리한다.

## M0. 프로젝트 초기화

- [x] Vite + TypeScript 초기화
- [x] Phaser 3 설치
- [x] 기본 게임 화면 출력
- [x] 1280×720 반응형 스케일 설정
- [x] 타입 검사 및 빌드 명령 구성
- [x] GitHub Pages 배포 워크플로 작성
- [x] README 실행 방법 갱신

완료 조건: 빈 게임 화면이 로컬에서 실행되고 프로덕션 빌드가 성공한다.

## M1. 최소 게임 루프

- [x] 메뉴 화면과 게임 시작
- [x] 플레이어 생성
- [x] WASD·방향키 이동
- [x] 일반 적 생성
- [x] 적의 플레이어 추적
- [x] 가장 가까운 적 자동 조준
- [x] 투사체 발사와 충돌
- [x] 적 처치
- [x] 플레이어 접촉 피해와 무적시간
- [x] 멘탈 HUD
- [x] 게임 오버와 다시 시작

완료 조건: 시작부터 사망과 재시작까지 플레이할 수 있고, 적과 투사체가 화면 밖에서 누적되지 않는다.

## M2. 성장 시스템

- [x] 경험치 아이템 드롭
- [x] 접근 시 경험치 자동 흡수
- [x] 레벨과 필요 경험치 계산
- [x] 레벨업 시 게임 정지
- [x] 무작위 능력 카드 3개 표시
- [x] 능력 선택과 게임 재개
- [x] 능력 중첩과 최대 레벨
- [x] 희귀도별 등장 확률
- [x] 초기 능력 8종 구현

완료 조건: 한 판에서 여러 번 레벨업할 수 있고 모든 능력 효과가 실제 능력치에 반영된다.
